Requerimientos Opcionales US 0012 Miguel Avalle Grupo 1




US 0012 A El usuario debe ser capaz de poder eliminar sus propios posts


Method
	SIGN
	DELETE
	/products/post?user_id={userId}&post_id={postId}
	Ejemplo: /products/post?user_id=1&post_id=1
	Response
	{
   "status": 200,
   "message": "operación exitosa"
}
	



Filtros/Parámetros


Parámetros
	Tipo
	Descripción/Ejemplo
	user_id
	int
	Número que identifica a cada usuario
	post_id
	int
	Número que identifica a cada post
